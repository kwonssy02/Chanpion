# Chanpion page
**찬피온**의 서버 소스코드입니다. (nodejs)


## Installation
npm 모듈과 필요 라이브러리를 설치합니다.
```console
sudo apt-get install npm
npm install
```


## To run with nodemon
```console
sudo nodemon ./bin/www
```


## To run with forever
```console
sudo forever start ./bin/www
```
